# Easy-Browser
An easy way to browse your favorite websites.
##Overview##
Embed Web Kit and learn about delegation, KVO, classes and UIToolbar.
###Learning Goals###
<ul>
	<li>WKWebView</li>
	<li> Delegation</li>
	<li> Classes + Structs</li>
  <li>NSURLRequest </li>
  <li> UIToolbar</li>
  <li>UIProgressView </li>
  <li>Key-value observing</li>
</ul>
